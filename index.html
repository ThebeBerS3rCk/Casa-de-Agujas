[ articulo / zine / pagina arrancada ]

La liturgia visual no debe sentirse digital. Debe sentirse manipulada.

Nocturna se presenta como si hubiera sido compuesta entre tijeras, cinta, toner y supersticion. El blanco nunca es puro. El negro nunca descansa.

Cada bloque editorial se apoya en una grilla rota, con margenes falsos, rotaciones minimas y paneles que se pisan entre si como si fueran recortes pegados durante la madrugada. Nada busca simetria impecable.

El resultado tiene que parecer una mezcla de fanzine, afiche ceremonial y pagina de grimorio moderno: brutal, ceremonial, fotocopiado, lleno de huellas y aun asi legible.

anotacion al margen

La interfaz se lee como un recorte recuperado de un archivo clandestino: ceremonial, tangible, incomodo y deliberadamente imperfecto.

[ resumen / demo / landing ]

Esta pagina funciona como una muestra visual estatica para presentar una marca ficticia con una identidad oscura y ritual.

El sitio combina un hero dramatico, un manifiesto tipo zine, un collage monocromo y un tablero simbolico para mostrar una direccion visual fuerte.

Todo esta construido con HTML, CSS y JavaScript vainilla, sin dependencias externas, con estructura responsive y funcionamiento offline.
